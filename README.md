## What You Get: Features Breakdown

Here's what's included regardless of which slot tier you pick:

**Instant Automated Setup** — The server spins up the moment you order. No waiting around for a human to configure things on the backend. You get login credentials immediately and can be in-game within minutes.

**TCAdmin Control Panel** — This is a well-known, battle-tested control panel in the game hosting world. It's not the flashiest UI but it gets the job done. From the panel you can restart your server, run Steam updates, edit config files, manage files via the built-in file manager, and schedule tasks.

**Oxide / Plugin Support with 1-Click Install** — Oxide (now uMod) is the modding framework that powers most Rust plugins. BlackBox lets you install it with a single click, and there's a mod manager GUI so you're not fumbling around with FTP just to add a home teleport plugin. Auto-updating mods are also supported.

**Automatic Offsite Backups** — Every time you stop or restart your server, it automatically zips up your important files and ships them off to an offsite backup server. This is legitimately useful — Rust world files aren't tiny, and losing a few weeks of progress to a hardware failure is the kind of thing that breaks up friend groups.

**Free DDoS Protection** — Rust servers get hit with DDoS attacks more often than you'd expect, especially if your server gets popular or someone has a grudge after getting raided. Having this built in at no extra cost matters.

**Free Debranding** — Small thing, but nice: they'll remove BlackBox branding from your server so it looks like "your" server, not some third-party host's property.

**Easy Admin Management** — You can add admins by pasting in a Steam ID and clicking save. No command-line nonsense required.
